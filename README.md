# Wedding Invitation App — Reel Style

Luxury Indian wedding invitation inspired by Instagram reel invites: envelope opening, full-screen scroll slides, cream & maroon theme.

## Features

- **Tap-to-open envelope** with wax seal animation
- **Reel-style scroll** — each section fills the screen (like Instagram)
- **Cinematic text reveals** on couple names
- **Save the Date** with photo zoom effect
- **Swipeable event cards**
- **Gallery** with lightbox
- **RSVP form**
- **Music toggle** (add `public/wedding-music.mp3`)

## Customize

Edit **`src/lib/wedding-data.ts`** for names, dates, events, photos.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000 — **tap the envelope** to start.

## Music

Place an MP3 at `public/wedding-music.mp3` for background music on the top-right button.
